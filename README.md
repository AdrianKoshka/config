# Config
Various config files

## Structure

### `.config`

```
.config
├── Code
│   └── User
│       └── settings.json
├── i3
│   └── config
├── mpd
│   └── mpd.conf
└── sytemd
    └── user
        ├── discord.service
        └── znc.service
```

### `/etc`

```
etc
└── systemd
    └── system
        ├── bitlbee.service
        ├── bitlbee.service.d
        │   └── override.conf
        ├── mpd.service.d
        │   └── override.conf
        └── unifi.service.d
            └── override.conf
```
